# KH_SEMI_CLASS2
Love Diary 커플 다이어리

## 🖥️프로젝트 소개
Love Diary는 커플들이 일상을 기록하고 공유할 수 있는 다이어리 어플리케이션입니다.
커플들이 서로의 일상을 기록하고 추억을 쌓는 한편 앞으로의 계획을 세우고 공유할 수 있는 어플리케이션으로 기획하였습니다.
